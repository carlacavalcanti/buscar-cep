# Buscador de CEP

Projeto desenvolvido em React consumindo a API ViaCEP.

## Funcionalidades

* Buscar endereço através do CEP
* Consumir API externa
* Exibir logradouro, bairro, cidade e estado
* Interface estilizada com CSS

## Tecnologias utilizadas

* React
* JavaScript
* CSS3
* API ViaCEP

## Como funciona

O usuário digita um CEP no input e o sistema faz uma requisição para a API ViaCEP utilizando `fetch()` e `async/await`.

Depois disso, os dados retornados são exibidos na tela.

## Conceitos praticados

* useState
* Eventos no React
* Formulários controlados
* API REST
* fetch
* async/await
* JSON
* Template Strings

## Como executar o projeto

Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
